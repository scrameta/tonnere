STM32F407ZG (U19) CubeMX-style summary and pinlist

This is derived from the KiCad netlist (tonnere.net) and reflects which STM32 pins are wired to which functions.

Peripheral summary

  • **FMC/FSMC**: AF12, external bus signals (`/FPGA/FSMC.*`) – address/data/control to FPGA-side FSMC client.
  • **I2C**: PB8/PB9 (AF4) + an extra GPIO reset line (PC5).
  • **SPI**: PB4/PB5 (AF5) + two chip selects (PB6/PB7 as GPIO outputs).
  • **USART**: PB10..PB14 correspond to **USART3 (AF7)** (even if the net label says USART1).
  • **UART to ESP32**: PC6/PC7 correspond to **USART6 (AF8)**.
  • **GPIO handshaking to ESP32**: PG8/PG9/PG10 are GPIO outputs to ESP32 input-only pins (`ESP.SGPI0..2`), and PA9 is a GPIO input from ESP32 (`ESP.SGPIO`).
  • **Analog**: joystick/paddle pots + audio ADC lines are set up as analog inputs.
